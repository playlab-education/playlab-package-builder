# Playlab Sales Hub

Single-page HTML app (index.html) serving as the Playlab Sales Hub — includes a package builder, pricing reference, key resources, and a welcome landing page. No build step, no framework — vanilla JS with embedded CSS. Deployed via GitHub Pages from `main` branch.

## Tabs
- **Welcome** (default) — Landing page linking to other tabs (`#welcomeView`)
- **Builder** — Package builder for service quotes (`.builder`)
- **Pricing Sheet** — Reference pricing tables (`#pricingView`)
- **Key Resources** — Embedded Google Docs/Sheets/Slides (`#resourcesView`)
